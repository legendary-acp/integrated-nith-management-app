# NITH-Management
This repository contains a app that helps in bringing some of the management tasks of NITH online. This app is designed using flutter framework and is still under construction. Feel free to contribute ðŸ˜‰.

## Tasks automated by App

The website aims to achieve automation of following task in its initial phase:
- Students Attendance Management
- Online Quiz
- Result Display and Supplementary Examination Registration System
- Timetable and exam date sheet management system
- Hostel Allotment System
- Hospital Management System
- Hostel Complaint System
